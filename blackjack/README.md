# Blackjack C Game 🃏

A simple, terminal-based **Blackjack (21)** game implemented in C. 

## Features
- Dynamic **Ace logic** (Values of 1 or 11 based on hand total).
- Automated Dealer AI (Stays at 17).
- Professional coding standards (Clean code, English naming).

## How to Run
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/Blackjack-C-Game.git`
2. Compile: `gcc src/main.c -o blackjack`
3. Run: `./blackjack`
