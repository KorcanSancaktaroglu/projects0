# ✊ Rock Paper Scissors - C Edition ✂️

A lightweight and structured implementation of the classic **Rock, Paper, Scissors** game in C. This project demonstrates modular logic, randomized AI moves, and a scalable architecture for game session tracking.



## 🚀 Key Features
* **Smart Input:** Automatically handles both lowercase and uppercase (R, P, S).
* **Randomized AI:** Computer moves are truly unpredictable using `time(0)` seeding.
* **History Ready:** Includes a `GameHistory` linked-list structure for future score tracking.
* **Robust Logic:** Clean separation between game engine and main execution.

## 🛠️ Technical Stack
* **Language:** Pure C
* **Core Concepts:** Structs, Randomization, Conditional Logic, Character Normalization.
